ToDoTrack

Background

I started working on this project to gain experience with the Laravel framework. There is currently not much in to see but it could be updated occasionally.

About the Project

ToDoTrack will be a simple ToDo list application that allows you to track the time you worked on one task.

At the time there is only logic implemented with some tests, and it is currently not usable with a browser.

Technology Stack

  • Laravel 5.2
  • PHP 7
  • HTML5
  • Bootstrap
  • jQuery
