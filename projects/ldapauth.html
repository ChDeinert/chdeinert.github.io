LDAPAuth

About the Project

This Project was created while I was working for MCQ TECH GmbH . With a potential of hundrets of users accessing the Intranet applications, that are already registered in Active Directory, a way to Authenticate these users against the AD was needed.

Since there was no module for the Zikula Application Framework 1.3.x that enabled only the Authentication with LDAP, this one was created. It allows users - that have an account on the AD - to log in without the need of a registration. Administrators can also import multiple users and groups into Zikula and update the imported informations about multiple users.

Another feature is the possibility to automatically update the informations in Zikula Profile module.

The connection to the Active Directory happens with the help of the PHP Package adLDAP . If the connection to the Active Directory fails for some reason, users that successfully logged in at least once, are still able to since the login informations are also saved in the standard Zikula account.

Since much of the development of this project was done during my work for MCQ TECH, the further development is currently on a halt. Until I find the muse and the time to build a working test-environment and work on it.
